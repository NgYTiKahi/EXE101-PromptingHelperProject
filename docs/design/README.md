# UI/UX Design

This folder contains the UI/UX design documents and mockups for PromptAds Studio.

PromptAds Studio is a guided prompt builder for AI advertising images and videos. The design focuses on simplicity, clarity, and speed for non-technical users.

## Main Screens

1. Home Page
2. Prompt Builder Page
3. Generated Prompt Page

## User Flow

Home Page → Prompt Builder → Generated Prompt → Copy / Regenerate / Save / Export

## Screen Descriptions

### Home Page

The Home Page introduces PromptAds Studio and explains the main value of the product: helping users build AI advertising prompts quickly.

### Prompt Builder Page

The Prompt Builder Page collects user input, including product information, target audience, platform, content type, advertising style, main message, language, and call to action.

### Generated Prompt Page

The Generated Prompt Page displays the final AI advertising prompt. Users can copy, regenerate, save, or export the prompt.

## Design Goals

- Help users create advertising prompts quickly
- Use a guided form instead of manual prompt writing
- Make the interface simple for non-technical users
- Provide clear actions such as Generate, Copy, Save, and Export
- Support advertising content for images and videos

## Target Users

- Small online shops
- Freelancers
- Students
- Content creators
- Small businesses

## UI Style

- Modern SaaS-style interface
- Dark navy background
- Purple accent color
- Rounded cards
- Clear call-to-action buttons
