# User Flow

1. User opens the Home Page.
2. User clicks "Start Creating Prompt".
3. User fills in product information.
4. User selects the target audience, platform, content type, and advertising style.
5. User enters the main message, language, and call to action.
6. User clicks "Generate My Prompt".
7. The system displays a complete AI advertising prompt.
8. User can copy, regenerate, save, or export the prompt.
9. User uses the prompt in another AI tool such as ChatGPT, Gemini, Midjourney, DALL-E, Sora, or an AI video generator.
